# Dora-bolos
🎂 Dora Bolos  Este projeto foi criado para apresentar a Dora Bolos na internet de forma simples, bonita e acolhedora. A ideia foi transmitir o cuidado e a tradição de uma confeitaria que há mais de 20 anos prepara bolos artesanais por encomenda, valorizando cada detalhe e facilitando o contato com novos clientes.
